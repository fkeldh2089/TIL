1. CSS flex-direction
   - row : 좌에서 우로 정렬
   - row-reverse : 우에서 좌로 정렬
   - column : 상에서 하로 정렬
   - column-reverse : 하에서 상으로 정렬



2. Bootstrap flex-direction
   - .flex-row
   - .flex-row-reverse
   - .flex-column
   - .flex-column-reverse



3. align-items
   - stretch : 구성요소의 여백이 해당 구역만큼 늘어나며 배열, 컨테이너를 채움
   - flex-start : 시작(위)부터 배열
   - flex-end : 끝(아래)부터 배열
   - center : 가로축 중앙에 배열 오버플로 가능
