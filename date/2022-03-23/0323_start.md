## 연습문제 1

> 7개의 byte -> 7개의 bit
> 2진수를 10진수로 만드는 방법

![Snipaste_2022-03-23_03-09-01](0323_start.assets/Snipaste_2022-03-23_03-09-01.png)

```sh
# input
2
00000010001101
0000000111100000011000000111100110000110000111100111100111111001100111
```

```sh
# output
1 13
0 120 12 7 76 24 60 121 124 103
```
