# TCP/IP 4계층

- OSI 7계층보다 먼저 나온 규격이지만 ,현재 더 많이 사용되고 있다.

|      |    Layer     |                           Content                            |
| :--: | :----------: | :----------------------------------------------------------: |
|  4   | application  |             응용 계층, 표현 계층, 세션 계층             |
|  3   |  transport   |           전송 계층           |
|  2   |   ineternet    |   네트워크 계층   |
|  1   |     network Interface     | 물리 계층과 데이터 링크 계층 |
