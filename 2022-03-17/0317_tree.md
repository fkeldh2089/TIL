- subtree

- 이진탐색

  - 트리의 표현 방법
  - 해당 노드에 값을 입력하는 방법, 중위 순회를 해야 하는 이유

  - 위 2가지 조건을 명확히 한다면 어렵지 않게 풀 수 있는 문제

- 이진 힙

  - 순서대로 입력된 값을 힙 조건에 맞게 트리로 표현 하는 법을 필요로 함

  - 탐색 위치부터 부모 노드를 찾아가는 방법 

  - `자식 노드의 노드 번호 // 2` 는 부모 노드의 노드번호
